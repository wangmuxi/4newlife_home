# 4newlife_home

精致的 `4newlife.top` 导航主页，使用 Next.js 15、Tailwind CSS 和 lucide-react 构建。

## 导航区块

- 我的生态：`blog.4newlife.top`、`verse.4newlife.top`、`quickhelpguide.com/meetschedule`
- 圣经资源：`wd.bible`、`biblegateway.com`、`微读书城`
- AI生产力：`Google AI Studio`、`OpenRouter`、`Hugging Face`

## 开发

```bash
npm install
npm run dev
```

## 验证

```bash
npm run typecheck
npm run build
```
